# Сайт «Ликвидация водопритока в скважинах» (по образцу Poweltec)

Трёхъязычный сайт нефтесервисной компании: **украинский** (основной), русский, английский.
Наполнение пока служит образцом (взято с poweltec.com) и будет заменено собственным.

## Как открыть

Откройте файл `index.html` в любом браузере — сайт полностью автономный,
интернет нужен только для загрузки шрифтов.

## Из чего состоит

| Файл / папка | Что это |
|---|---|
| `index.html` | Главная страница |
| `technology.html` | Технологии (с двумя анимациями) |
| `offers.html` | Услуги |
| `library.html` | Библиотека публикаций и презентаций |
| `clients.html` | Отзывы клиентов |
| `quality.html` | Качество (ISO 9001, экология) |
| `about.html` | О компании и лаборатории |
| `contact.html` | Контакты |
| `css/style.css` | Все стили: цвета, шрифты, отступы, мобильная вёрстка |
| `js/main.js` | Переводы (украинский/английский) и вся интерактивность |
| `js/library-data.js` | Список публикаций библиотеки |
| `assets/img/` | Фотографии (источники и лицензии — в `CREDITS.txt`) |
| `assets/anim/` | Анимации со страницы «Технологии» |
| `presentations/` | Сюда кладите PDF-файлы своих презентаций |
| `docs/` | Сюда кладите брошюру и сертификаты (PDF) |

## Как всё редактировать

Весь код снабжён подробными комментариями на русском языке — открывайте
любой файл в текстовом редакторе (например, «Блокноте») и читайте
пояснения в рамках `<!-- ... -->` (HTML) или `/* ... */` (стили и скрипты).

- **Изменить русские тексты** — правьте их прямо в HTML-файлах страниц.
- **Изменить украинские/английские тексты** — правьте словари в начале `js/main.js`.
- **Добавить презентацию** — положите PDF в `presentations/` и допишите
  одну запись в начало списка в `js/library-data.js` (образец в шапке файла).
- **Заменить фото** — положите новую картинку с тем же именем в `assets/img/`.
